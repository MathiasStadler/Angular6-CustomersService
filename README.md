## TL;DR

```bash
brew install mongodb
mongod --dbpath /tmp/mongo-data
# open another shell
git clone <REPO>
cd <REPO_DIR>
cd client
npm i
cd ../server
npm i
start_prod_jit
# open a brwoser and call http://localhost:3000
```
